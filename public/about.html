Overview

This project aims to develop a web-based application for real-time temperature monitoring of chemical solutions involved in construction of insulators.
It automates data collection process and allows users to access and visualize temperature data in a convenient and accessible manner.

Problem Statement

The manual monitoring of the 100-hour boiling test requires continuous human presence, which is time-consuming and prone to errors. A web-based mobile application that displays real-time temperature data can automate the monitoring process and improve efficiency.

Implementation

Hardware

The below listed equipment is used to construct the edge device responsible for data collection, pre-processing and transmission to cloud data centers.

  • Processor: Raspberry Pi 4 Model B (Quad-coreCortex-A72 64-bit SoC @ 1.5GHz)
  • Memory: 2GB LPDDR4 SDRAM
  • Storage: 32GB SD microSD Card
  • Temperature Sensor: DS18B20 Digital Temperature Sensor

Software

Following are the tools used in building the software web-based platform, and establishing the cloud connection for data storage.

  • Operating Systems:
    • Windows 10/11 for web-platform development.
    • Raspberry Pi-OS, formerly Raspbian for edge device programming.
  • Programming:
    • Python for programming Raspberry Pi-4.
    • Javascript for programming the web-platform.
  • Development Environment: Visual Studio Code IDE
  • Libraries and Frameworks:
    • Express for server-side programming.
    • Axios for HTTP request handling during Cloud communication.
  • Cloud Services: Thingspeak Cloud
  • Security:
    • SSL/TLS for secure communication.
    • API key authentication for data channel access.

Circuit Diagram

This is the circuit diagram for the data-collecting edge-device. It uses DS18B20 Temperature Sensor for temperature data collection, which is pre-processed at edge level by the Single Board Computer Raspberry Pi-4.
After the data is processed, it is then sent over to Thingspeak Cloud Channel for visualization and possibly aid in data-analytics.
